# Saúde do Futuro - Covid-2019


## Dependências
- [Jekyll](https://jekyllrb.com/docs/installation)
- [Bundler](https://bundler.io/)

## Como configurar?

```bash
./scripts/bootstrap   
```

## Como rodar?

```bash
bundle exec jekyll serve
```

Depois disso, acesse a aplicação em `http://127.0.0.1:4000`
